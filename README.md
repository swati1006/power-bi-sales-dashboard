
# 📊 Power BI Sales Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard built using AdventureWorks data to analyze sales performance, customer behavior, and product trends. The dashboard provides clear insights to support data-driven decision-making.

---

## 🛠 Tools Used
- Power BI Desktop  
- Microsoft Excel  
- Data Cleaning & Visualization  

---

## 📊 Dashboard Preview

### 🔹 Executive Dashboard

This dashboard provides a high-level overview of business performance:
- KPI cards showing **Total Revenue, Profit, Orders, and Return Rate**
- **Sales trend over time** using a line chart
- **Orders by category** to identify top-performing segments
- **Top products table** with revenue and return analysis

---

### 🔹 Detail Dashboard

This dashboard focuses on customer insights:
- **Customer growth trend** over time
- **Top customers by revenue**
- **Orders distribution by income level**
- **Orders by occupation** to understand customer segments
- Identification of **top-performing customers**

---

---

## 📈 Key Insights
- Identified top-performing product categories driving revenue  
- Analyzed sales trends to track business growth  
- Evaluated customer segments based on income and occupation  
- Monitored return rates to assess product performance
